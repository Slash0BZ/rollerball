Introduction

This project is about a rollerball scene simulation that users can build rails then let a ball fall on the rail.

We used WebGl for this project, so all the code is in html and javascript.

Project started since July 26 2016 and estimation of finishing is August 12 2016.

Milestone 1

July 26 - August 2

In thie mile stone, we finished the following goal as discussed in Proposal:

Silei part: physical system and view control module.

Xuanyu part: ball and rail construction, phong shader and project site construction

Comment: The development of this project is smooth, we finished everything we discussed in the proposal earlier.
We used a small WebGL library Lightgl.js, which helped us with lower-level WebGL details like buffer,
and front end library Bootstrap which helped us with this project site.
Librarys listed above included all the code that are not written by us but used in this project. Everthing in view.js and move.js is fully implemented by us.
At this time, we only implemented a standard half circle as rail, but our structrual design and most of the implementations are expandable.
In the next milestone, we will focus on arbitrary expansions on some modules that we have not solved yet.
